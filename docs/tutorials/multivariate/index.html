Multivariate Statistics

Dimensionality reduction, clustering, and classical multivariate methods for data with many correlated variables

When data contain many variables measured on the same units – gene expression across thousands of transcripts, questionnaire batteries with dozens of items, physiological measurements across multiple domains – univariate methods are rarely sufficient. Multivariate statistics provides tools for reducing dimensions, discovering structure, classifying observations, and summarising relationships between sets of variables.

This area covers classical methods that remain the default first pass for most multivariate datasets, alongside their modern refinements.

Methods covered

  • Principal component analysis (PCA): derivation, scaling choices, number-of-components selection, biplots
  • Factor analysis: exploratory and confirmatory, rotation (varimax, promax, oblimin), goodness-of-fit
  • Multidimensional scaling (MDS): metric and non-metric, Shepard diagrams
  • Correspondence analysis for categorical data and its multiple-variable extension
  • Cluster analysis: hierarchical (agglomerative and divisive), k-means, k-medoids (PAM), DBSCAN
  • Cluster validation: silhouette, gap statistic, elbow heuristics, stability by resampling
  • Linear and quadratic discriminant analysis
  • Canonical correlation analysis: relating two sets of variables
  • Partial least squares (PLS) regression and PLS-DA
  • Distance and dissimilarity measures: Euclidean, Mahalanobis, Gower, Jaccard, Bray-Curtis

Packages used throughout include stats, FactoMineR, factoextra, cluster, vegan, MASS, psych, and mixOmics.
