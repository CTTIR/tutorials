# DOE Lab

Generate full factorial, fractional factorial, Plackett-Burman, central
composite, and Box-Behnken designs interactively. Displays the run
table, alias structure, and a correlation heatmap of design columns.

```r
shiny::runApp("apps/16-design-of-experiments-lab")
```

Requires: `FrF2`, `rsm`, `DT`.
